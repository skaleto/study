spring九大核心组件：
MultipartResolver
    多文件上传组件
    
LocaleResolver
    国际化组件
    
ThemeResolver
    页面定制皮肤更换风格的主题模板组件
    
HandlerMappings
    保存url和method映射关系的组件
    
HandlerAdapters
    动态参数匹配组件

HandlerExceptionResolvers
    异常拦截器

RequestToViewNameTranslator
    视图提取器

ViewResolvers
    视图转换器

FlashMapManager
    参数转换器
